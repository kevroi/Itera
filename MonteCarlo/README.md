The script `nSpehre.py` computes the volume of a sphere of radius r, living in n+1 dimensions (mathematically known as an n-sphere, S<sub>n</sub>).

S<sub>n</sub> = { xϵℝ<sub>n+1</sub> : ||x|| = r }
